# 外包交接清单

- [ ] Git 仓库包含完整可读源码和锁文件，无仅构建产物交付。
- [ ] Storybook 可本地和预览环境运行。
- [ ] 提供环境变量说明，但不包含任何真实密钥。
- [ ] 提供 Mock 数据、真实 API 切换方式和联调记录。
- [ ] 提供路由、页面、组件和 API 映射表。
- [ ] 提供单元、E2E、可访问性和性能报告。
- [ ] 所有已知问题均有复现、影响和负责人，不以口头说明替代。
- [ ] 外包自建临时服务、账号和第三方依赖已移交或移除。
- [ ] 完成部署、回滚和故障排查演示。
- [ ] 甲方能够在干净环境独立安装、测试、构建和启动。
