# 页面验收清单

- [ ] 页面有一个清晰视觉中心和一个主操作。
- [ ] 首屏能回答“做什么、在哪里做、看哪里、下一步点哪里”。
- [ ] 当前作品/内容区域在桌面端占主要空间，高级参数默认收起。
- [ ] 直接 URL、刷新、前进后退都能恢复当前项目、课时和步骤。
- [ ] 加载、空、错误、无权限、过期、部分成功和取消态均有真实界面。
- [ ] 生成任务离开页面后继续，返回后从后端恢复。
- [ ] 提示词可见且可编辑区与平台锁定区清楚区分。
- [ ] 用户编辑不会因刷新、失败或 SSE 重连丢失。
- [ ] 键盘、焦点、对比度、表单标签和错误提示满足 WCAG 2.1 AA。
- [ ] 1440、1280、1024 和窄屏断点无横向溢出或主操作遮挡。
- [ ] 不显示模型网关、节点运行等不必要专业术语给普通教师。
- [ ] 埋点、错误追踪和性能指标不采集敏感正文。
