# 组件验收清单

- [ ] 组件职责单一，业务组合放在 feature 层。
- [ ] 没有复制 API DTO、状态枚举或节点映射。
- [ ] 无硬编码十二字段、供应商模型或特定项目 ID。
- [ ] 受控/非受控模式、加载、禁用、错误和只读状态明确。
- [ ] 交互组件可键盘操作，焦点可见，语义和 ARIA 正确。
- [ ] 颜色、间距、圆角、阴影和动效来自令牌。
- [ ] 异步组件处理取消、重复点击、幂等和竞态。
- [ ] Storybook 覆盖主要状态和长中文内容。
- [ ] 单元/交互测试覆盖关键行为，不只做快照。
- [ ] 巨型组件已按布局、领域和交互拆分，副作用集中在 hooks/service。
