# 视觉系统与动效

## 1. 视觉定位

主题为“山海创作空间”：温暖、清晰、有作品感、有教育气质、专业但不压迫。主页具有品牌表现，功能页装饰退后，让作品成为视觉中心。

## 2. 色彩和背景

- 山海蓝：主按钮和关键选中。
- 深海蓝：标题和专业场景。
- 天空蓝：轻提示和进度。
- 暖白：全局背景。
- 纯白：教案和PPT正文画布。
- 绿、琥珀、红、灰只用于完成、等待、失败和未启用状态。

主页允许克制浅蓝渐变和淡山海纹理；项目使用暖白；PPT使用中性工作区；视频播放器可深色；管理端使用白和浅灰。不得把所有页面做成相同白卡片列表。

## 3. 组件视觉

- 普通卡片圆角12px，大作品卡16px，按钮和输入8—10px。
- 卡片间距16—24px，主要区块32—48px。
- 普通卡片弱阴影，当前作品轻微浮起，抽屉使用柔和明显阴影。
- 禁止三层卡片嵌套、处处边框、多个高饱和主按钮、重渐变和玻璃拟态。

## 4. 字体

主页标题36—44px，页面标题24—28px，任务标题20—24px，卡片16—18px，正文14—16px，教案正文16px并使用舒适行高。中文使用清晰无衬线字体并提供系统字体回退，不依赖不可控外部字体CDN。

## 5. 动效

- 按钮100—140ms；卡片160—200ms；面板200—260ms；页面切换240—320ms；候选放大280—360ms。
- 动效用于候选切换、面板展开、任务阶段、保存回项目、PPT页面和视频镜头切换。
- 禁止背景持续漂浮、按钮发光、无限旋转Logo和频繁弹跳。
- 遵守`prefers-reduced-motion`。

## 6. 加载和空状态

短请求使用骨架屏。长任务显示真实阶段，不伪造进度。空状态说明为什么为空、下一步做什么，并提供明确动作。

## 7. 业务组件

统一设计`FocusPageHeader`、`CreativeCanvas`、`ContextDrawer`、`DirectEditToolbar`、`CandidateViewer`、`CreationInput`、`TaskStatusBar`、`ProjectResultCard`、`ContentDefinitionRenderer`、`AssetPicker`、`VersionHistory`和`SaveToProjectDialog`。
