# 项目与课时工作台

## 1. 项目内导航

```text
项目总览｜教材与课时｜课时工作台｜素材与成果｜项目任务｜项目交付
```

项目总览以继续工作为核心，展示教材、课时卡、分支状态、待确认作品、运行任务和当前成果。PPT和视频必须区分未启用、已跳过、制作中、等待确认、已完成和内容已变化。

教案正文、课堂导入设计和视频在课时工作台中分别建模。PPT未取得批准教案时显示前置条件；视频只等待教师选中一套导入方案，不等待教案。

## 2. 课时工作台布局

顶部项目上下文栏约56px，只保留项目、课时、执行模式、保存状态、费用摘要和暂停操作。

左侧流程栏200—240px，可折叠：

```text
准备课程
  查看教材
  安排课时

写教案
  生成教案
  修改并确认

选课堂导入
  查看三类九套
  选择一套方案

做PPT
  安排页面
  设计封面
  制作正文
  导出PPT

做导入视频
  编写母版剧本
  安排故事镜头
  确定画面风格
  制作镜头图片
  制作视频片段
  合成完整视频

下载成果
```

中央作品区默认占可用空间70%以上。右侧调整面板默认收起，点击参考内容、生成要求、进一步调整、检查结果或历史记录后展开。底部任务默认收成状态条。

## 3. 作品直接编辑

- 教案：点击章节和教学环节直接编辑；选中文字可改写、精简或补充。
- 导入设计：先展示推荐度打分和九张方案卡；展开可编辑独立创意、最小课程锚点、课堂首问、交接时刻和不得提前讲授内容。明确提示“选择不影响教案通过，只有视频需要此选择”。
- PPT大纲：逐页展示唯一教学任务、来源和页型，可拖动排序但必须提示知识顺序影响。
- PPT封面：大画布比较候选，采用后明确显示“正文将继承此视觉语言，背景仍为纯白”。
- PPT正文：白色16:9画布直接编辑；点击文字和数学图形修改，点击图片主视觉可替换、重新生成或从项目选择。右侧按需展开“画布、图片、文字、排版”四层详情。
- 母版剧本：使用沉浸式文档编辑器，场次、画面动作、旁白、对白和交接时刻同页编辑。
- 粗分镜：使用全宽故事节拍时间线。
- 细分镜：点击镜头卡直接修改画面、动作、时长和声音意图。
- 视频候选：点击即播放、比较和采用。

基础修改不得要求用户先寻找右侧表单。

## 4. 当前任务与主操作

每页标题先说明当前要完成什么，并只保留一个主按钮。例如封面页：

```text
当前要做：选择一张PPT封面
选择后，系统会按照这张封面的风格制作正文

主操作：采用这张封面并继续
```

重新生成、提出修改和比较版本是次级操作；下载和生成记录进入更多菜单。

## 5. 导入创作中心

项目节点只展示待生成数量、共享画面风格和准备状态：

```text
需要生成12张镜头图片
已准备画面风格、参考图和12条生成要求

检查待生成内容｜去图片创作台
```

不得在项目节点复制完整生图或视频工作台。

## 6. 响应式

| 宽度 | 规则 |
|---|---|
| ≥1440 | 左侧、中央和按需右侧同时可用 |
| 1280—1439 | 左侧缩窄，右侧展开时中央自适应 |
| 1024—1279 | 右侧使用抽屉 |
| 平板 | 流程和调整均使用抽屉，支持审核和轻编辑 |
| 手机 | 查看任务、审核、播放和下载 |
