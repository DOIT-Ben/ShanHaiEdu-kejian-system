# 08 模型网关前端规格

## 1. 定位

模型网关是所有文本、图片、视频和TTS调用的统一管理入口。前端只提供配置、查看和操作界面；能力匹配、密钥存储、正式路由、预算判定、自动回退和费用结算均由后端执行。

## 2. 管理端页面

### 总览

- Provider健康状态。
- 今日调用次数、成功率、平均耗时和费用。
- 失败任务。
- 限流和降级状态。
- 各能力当前主模型。

### Provider管理

字段：名称、类型、Base URL、鉴权方式、密钥状态、超时、重试、并发、限流、环境和启用状态。  
操作：新增、编辑、连接测试、启用、停用和查看调用记录。

前端不得重新显示完整密钥。密钥输入提交后立即清空，只显示脱敏状态和更新时间。

### 模型目录

- 业务名称和真实模型名。
- Provider。
- 能力类型。
- 输入输出模态。
- 文本长度、图片尺寸、视频时长和参考图限制。
- 异步、取消、种子等能力。
- 价格和计费单位。
- 推荐场景和状态。

### 路由策略

- 能力对应的主模型。
- 备用模型顺序。
- 质量、成本、速度、平衡或指定模型模式。
- 机构和项目适用范围。
- 最大费用和参数边界。
- 是否允许跨Provider回退。

保存前必须显示路由模拟结果和冲突校验。

### 预算额度

- 平台日预算。
- 机构月预算。
- 教师额度。
- 项目预算。
- 单节点费用上限。
- 超额后的暂停或授权策略。

### 调用记录

筛选：日期、机构、项目、课时、节点、能力、Provider、模型、状态和是否回退。  
详情：输入摘要、PromptVersion、参数、任务ID、耗时、费用、错误和回退链。  
普通管理员不得查看敏感教材全文和完整提示词，除非拥有明确审计权限。

## 3. 教师端模型设置

普通模式显示：推荐、高质量、经济和快速草稿。  
高级模式在授权范围内显示真实Provider和模型。  
生成前显示候选数量、参数摘要、预计费用和备用模型状态。

教师不能：

- 输入API Key。
- 选择机构禁止的模型。
- 突破模板参数边界。
- 绕过预算授权。
- 强制对已经提交的付费任务进行静默回退。

## 4. 连接测试

点击测试后：

1. 前端提交测试命令。
2. 后端创建测试任务。
3. 前端通过轮询或SSE显示DNS、鉴权、模型可用性和最小请求测试结果。
4. 不把密钥或上游原始敏感响应返回前端。

测试结果包含测试时间、环境、耗时、结果和脱敏错误。

## 5. 回退展示

文本未形成有效输出时可显示自动切换备用模型。  
图片回退后所有候选按模型来源分组。  
视频已经提交付费任务后，重新使用备用模型必须显示二次费用确认或符合已授权的自动策略。

前端必须区分：

- 技术重试。
- 内容返修。
- Provider回退。

## 6. 权限

- 教师：查看项目允许的业务模型和费用估算。
- 模板管理员：配置模板推荐能力和参数范围，不管理密钥。
- 系统管理员：管理Provider、模型、路由和预算。
- 审计管理员：查看脱敏调用记录和变更日志。
