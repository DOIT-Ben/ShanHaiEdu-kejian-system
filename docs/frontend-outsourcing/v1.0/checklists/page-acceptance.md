# 页面验收清单

- [ ] 页面具有明确标题、当前项目和当前课时上下文。
- [ ] 正常、空、加载、错误、无权限和无数据状态完整。
- [ ] 长中文、长项目名和长错误文案不破坏布局。
- [ ] 1440、1280和1024宽度通过验收。
- [ ] 刷新和深链接恢复当前位置。
- [ ] 浏览器前进和后退正常。
- [ ] 键盘可到达全部核心操作。
- [ ] 焦点状态清晰。
- [ ] 页面没有控制台错误和未处理Promise。
- [ ] 页面不直接调用fetch。
- [ ] 服务端数据不复制到全局Zustand。
- [ ] 付费操作前显示预计费用。
- [ ] 错误包含解决动作和Trace ID。
- [ ] SSE断开后显示恢复状态。
- [ ] 已批准版本不被原地覆盖。
- [ ] 截图或视觉回归与批准Figma一致。
