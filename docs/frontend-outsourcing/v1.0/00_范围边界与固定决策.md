# 00 范围边界与固定决策

## 1. 本次外包包含

- 完整教师端生产前端。
- 完整管理端生产前端。
- Figma高保真设计与交互原型。
- 基础组件、复合组件和业务组件。
- OpenAPI生成客户端和统一错误处理。
- SSE实时任务、断线重连和轮询降级。
- 文件上传、预览、下载和任务进度。
- Mock模式、联调模式和生产构建。
- 单元、组件、契约和端到端测试。
- Storybook组件文档。
- 部署说明、源码、设计源文件和交接资料。

## 2. 本次外包不包含

- FastAPI后端实现。
- 数据库、Redis、Celery和对象存储实现。
- 模型Provider适配器。
- 提示词、教学模板和工作流规则的业务内容生产。
- 模型密钥保存和加密。
- 正式费用计算、最终模型路由和自动回退逻辑。
- 教材OCR、PPTX组装、FFmpeg剪辑和媒体转码。
- 生产服务器购买、模型充值和云账号开通。

## 3. 前后端边界

前端只访问版本化业务API：

```text
/api/v2/*
```

前端不得：

- 直接调用任何文本、图片、视频或TTS模型API。
- 保存完整API Key。
- 自行判断正式节点状态。
- 自行计算正式费用。
- 通过前端逻辑绕过预算、权限、审核或质量门禁。
- 将模型输出直接当作可信HTML执行。

## 4. 固定产品决策

- 第一版闭环小学数学，架构预留其他学科。
- 一个教材父项目可以创建多个课时子项目。
- 每个课时独立拥有教案、导入设计、PPT、视频和交付包。
- 教案必须使用十二部分结构。
- 三类九套是独立产物，不放进教案正文。
- 独立创意生成不读取教材、知识点、教案和PPT。
- 课程侧只能通过科普、应用、故事三类锚点合同连接独立创意。
- 视频生产只读取批准的导入方案版本、锚点版本和制作约束。
- 视频顺序固定为母版剧本、视觉方向、视觉母图、粗分镜、图片资产、细分镜、镜头提示词、视频片段、声音字幕和剪辑成片。
- 一个镜头允许多次生成，但只有一个当前批准片段。
- 提示词在前端可见、可编辑、可保存版本。
- 下游引用具体资产版本；上游变化只标记失效，不删除历史产物。
- 专业手动和半自动是主要模式；全自动只生成待审核草稿。

## 5. 固定视觉决策

- 视觉方向为“教育创作工作室”。
- 大面积暖白和纯白，山海蓝为主色。
- 不采用传统统计后台、玻璃拟态、重度渐变和暗色驾驶舱。
- 第一版不做全局暗色模式。
- PPT正文预览必须保持纯白背景。
- 重点适配1440×900和1280px宽桌面。
- 手机只提供查看、审批和预览，不承担完整制作。

## 6. 承接方可提出建议但不得自行改变

- 微文案润色。
- 图标细节。
- 空状态插画。
- 不改变结构的局部间距优化。
- 不改变业务合同的动效细节。

涉及页面结构、路由、业务状态、API、组件行为、颜色主体系、模型费用提示或审批机制的变化，必须走书面变更流程。
