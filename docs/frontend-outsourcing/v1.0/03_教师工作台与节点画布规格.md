# 03 教师工作台与节点画布规格

## 1. 桌面布局

基准画布：1440×900。最小完整桌面宽度：1280px。

```text
┌──────────────────────────────────────────────────────────┐
│ 项目/课时 │ 执行模式 │ 进度 │ 费用 │ 保存状态 │ 全局操作 │ 64
├────┬──────────┬─────────────────────┬─────────────┤
│全局│ 工作流   │                     │ 输入/提示词 │
│导航│ 节点导航 │     当前创作画布    │ 参数/校验   │
│72  │ 260-300  │                     │ 320-520     │
├────┴──────────┴─────────────────────┴─────────────┤
│ 生成任务坞：折叠48，展开约260                         │
└──────────────────────────────────────────────────────────┘
```

## 2. 顶部项目栏

- 项目切换器。
- 课时切换器。
- 执行模式。
- 总体进度。
- 预算与已用费用。
- 自动保存状态。
- 通知、帮助和用户菜单。

## 3. 全局导航

72px图标栏：首页、项目、资产、任务、交付、设置和管理端。  
所有图标必须有Tooltip和当前选中状态。

## 4. 工作流节点导航

分组：教材与课时、教案、导入设计、PPT、视频、交付。  
状态：未开始、可执行、排队、生成中、待审核、已批准、需返修、已失效、失败、阻断和跳过。  
折叠后保留当前阶段和阻断提示。

## 5. 中央创作画布

节点画布顶部固定：节点名称、状态、保存状态、主要操作和更多操作。  
节点画布必须通过注册表按产物类型渲染，不得在一个巨型页面组件中用连续条件判断实现。

### 教材证据查看器

- PDF页缩略图和页码定位。
- 提取文字与视觉描述。
- 证据验证状态。
- 未知项和质量问题。
- 教师纠正和批准。

### 课时划分编辑器

- 教材页码和课时卡双栏。
- 调整顺序、页码和内容边界。
- 合并、拆分、重新生成和版本比较。
- 批准后批量创建课时。

### 十二部分教案编辑器

章节目录固定包含十二部分。教学过程使用环节卡，支持教师活动、学生活动、关键问题、预期回答、评价证据、设计意图和板书变化。教案正文使用文档式编辑，不使用十二个孤立表单。

### 三类九套导入设计

科普、应用、故事三列，每类三套。每套必须明确分隔独立创意、类别锚点、课堂第一问、交接时刻和禁止提前讲授内容。支持单套返修、锁定独立创意、只重做锚点、复制自定义和批准。

### PPT工作区

子画布：大纲、页面脚本、图片资产、PPT预览和交付。封面显示完整设计；正文预览保持纯白。页面支持单页返修、替换图片、修改可编辑文字和批准关键样张。

### 视频工作区

固定顺序：母版剧本、视觉方向、视觉母图、粗分镜、图片资产、细分镜、镜头提示词、视频片段、声音字幕和剪辑成片。粗分镜使用时间线；细分镜使用镜头卡。每个镜头展示shot_id、时长、画面、声音、资产引用、候选片段和批准clip_id。

### 交付画布

显示交付清单、质量门禁、文件版本、生成时间、下载和重新打包。缺失或未批准产物必须明确阻断交付。

## 6. 右侧检查器

默认400px，可拖动至320–520px。标签：

- 输入。
- 提示词。
- 参数。
- 校验。
- 版本。
- 资产。

提示词标签必须区分可编辑创作提示词和只读运行契约。教师不能删除平台安全契约、输出Schema和资产白名单。

## 7. 底部任务坞

- 跨页面持续显示任务。
- 展示排队、进度、Provider状态、费用、失败和重试。
- 生成期间不阻塞整个工作台。
- 任务完成时更新正式节点状态并保留通知记录。

## 8. 响应式

- 1440及以上：完整四区布局。
- 1280–1439：缩窄节点栏和检查器。
- 1024–1279：检查器变为抽屉。
- 平板：允许查看、审核和轻量修改。
- 手机：仅查看任务、审批和预览交付物。
