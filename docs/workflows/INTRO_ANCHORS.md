# 三类九套课堂导入设计

## 1. 定位

三类九套是每课时默认生成的课堂导入方案库，作为独立、可替换、可单独审核的产物 `intro_option_set`。它不是教案十二部分中的固定第十三项，也不影响教案正文独立通过审核。

教师明确不需要课堂导入设计时可以关闭。教师已有明确创意时可以只完善该创意，不强制扩成九套。两条路径使用同一个方案集合同：`default_nine` 包含九套，`refine_existing` 包含一套。

## 2. 三种主要倾向

| 主要倾向 | 创意重点 | 典型课堂价值 |
| --- | --- | --- |
| 科普 `science` | 值得观察、解释或追问的现象、未知和变化 | 让学生带着观察结果或疑问进入学习 |
| 应用 `application` | 真实任务、限制、选择或完成目标的需要 | 让学生感受到知识点解决实际问题的必要性 |
| 故事 `story` | 人物或主体的目标、变化、阻碍、误会和悬念 | 用叙事推进形成明确的课堂交接问题 |

三种取向用于保证默认九套的覆盖，每种主要倾向各三套；它们不是互斥类别。每套还必须声明一至两个 `secondary_tendencies`，允许科普、应用和故事自然交叉。

## 3. 单节点课程驱动生成

`intro.generate_options` 在一次模型调用中直接生成最终 `intro_option_set`。课程依据从创意形成开始参与，不允许先生成与课程无关的内容，再在结尾贴入课题词。

节点只读取有效创意所需的最小课程种子：

- 已批准目标课时和稳定课时键；
- 本课知识点与一句话学习目标；
- 教学内容边界和 `must_not_preteach`；
- 年级、年龄段与教材证据摘要；
- 可选教师创意、媒介和时长偏好。

节点不读取完整教案、PPT或视频产物。默认路径生成九套，已有创意路径只完善一套；两条路径都直接输出最终方案，不产生中间创意产物。

## 4. 每套方案字段

| 字段 | 语义 |
| --- | --- |
| `option_key` | 稳定方案键 |
| `lesson_unit_key` / `knowledge_point` | 课时与知识点追溯 |
| `primary_tendency` / `secondary_tendencies` | 主要与辅助创作倾向 |
| `title` / `creative_concept` | 方案标题和从课程依据形成的创意概念 |
| `hook` / `viewer_value` | 开场钩子和观看价值 |
| `suggested_medium` / `duration_seconds` | 推荐媒介与时长 |
| `course_anchor` | 创意与本课内容的具体关联 |
| `classroom_first_question` | 结束后教师提出的第一问 |
| `handoff_moment` | 短片或活动停止、教师接回课堂的具体时刻 |
| `must_not_preteach` | 不得提前给出的定义、方法、结论和探究步骤 |
| `fit_reason` / `risks` | 适配理由与风险 |
| `recommendation_score` / `recommendation_reason` | 推荐分与依据 |

## 5. 选择协议

1. 前端展示按推荐度排序的方案卡和打分依据。
2. 教师可以修改允许字段并选择任意一套；人工指定记录为 `teacher_selected`。
3. 自动化策略允许时可以采用唯一最高分方案，记录为 `policy_default`。
4. 选择结果写入不可变 `intro_selection.snapshot`，不回写教案正文和方案集版本。
5. 教案正文和PPT不等待教师完成选择；视频分支必须等待选择。

## 6. 视频交接合同

视频的课程语义只来自被选方案不可变快照，其中包含方案键、课时键、知识点、创作倾向、`creative_concept`、`hook`、`viewer_value`、`course_anchor`、`classroom_first_question`、`handoff_moment`、`must_not_preteach`、媒介、时长和风险。

视频不得重新读取完整教材、教案或PPT，也不得重新选择或改写上游方案。母版剧本、粗分镜和细分镜可以继续读取各自已批准的视频内部产物，但最终镜头必须准确停在 `handoff_moment`。

## 7. 硬性质量门

- 默认九套的主要倾向各三套，至少一套包含两个辅助倾向。
- 每套都能追溯目标课时、知识点和教材证据，课程关联参与创意形成。
- 九套不是只换标题、角色或场景；三种倾向不得解释为互斥类别。
- 最高推荐分唯一，每套只有一个清晰的课程关联与交接时刻。
- 不提前讲出课堂需要学生发现的内容。
- 不因小学受众默认使用儿童、教师、教室或卡通角色。
- 方案中不包含分镜、旁白、字幕、资产清单、Provider参数或费用策略。
